Chapter 01.

怎麼安裝 LocWarp?

先從 GitHub Releases 下載 LocWarp 安裝檔,接著做三個前置(Apple USB driver、USB 信任、開發者模式),缺一項都會抓不到裝置。

Step 01 of 04

下載並安裝 LocWarp

GitHub Releases · NSIS 安裝檔

到 GitHub Releases 或 Google 雲端抓最新版的 Windows 安裝檔,檔名類似 LocWarp Setup X.X.XX.exe。兩個來源內容相同,哪邊快用哪邊。

GitHub Releases Google 雲端 專案首頁
Note · Windows SmartScreen
出現「Windows 已保護您的電腦」? 因為 LocWarp 未做 code-signing,第一次執行會被 SmartScreen 擋。點其他資訊 › 仍要執行即可繼續安裝。
Note · 需系統管理員
安裝 + 執行都需要 Admin 權限 LocWarp 的 WiFi Tunnel 功能需要建立 TUN 網路介面,因此安裝時會要求系統管理員,執行時也會跳 UAC。
Step 02 of 04

安裝 Apple USB driver

三種擇一: 桌面版 iTunes / Store 版 iTunes / Apple Devices

Windows 沒有內建 Apple 的 USB driver,必須透過下列方式之一安裝裝置驅動,之後 LocWarp 才能透過 USB 與 iPhone 溝通。

Note
三種擇一即可 多數用戶裝傳統桌面版 iTunes 就能用。若 iTunes (桌面版或 Microsoft Store 版) 兩個都試過 LocWarp 還是抓不到 iPhone,社群回報改裝 Microsoft Store 的 「Apple Devices」 可以成功。
桌面版 iTunes (64-bit) Microsoft Store iTunes Apple Devices (備案)
Step 03 of 04

USB 連接並信任電腦

建立 pair record

使用原廠 / 有資料傳輸能力的 Lightning / USB-C 線把 iPhone 接上電腦。iPhone 螢幕會跳出提示:

Prompt · iOS
「要信任這部電腦嗎?」 點信任,輸入 iPhone 解鎖密碼完成配對。這會在電腦端建立 pair record,後續 LocWarp 就可透過該配對與裝置通訊。
Tip · 疑難排解
沒跳提示? 檢查線材是不是充電專用線 (便宜副廠線常見);或打開 iTunes / Apple Devices 確認電腦有看到裝置。
Step 04 of 04

開啟開發者模式

iOS 16+ · 解鎖 DVT 協議

iPhone 上前往:

設定 › 隱私權與安全性 › 開發者模式

把開關打開,裝置會要求重啟。重啟後再次確認「啟用開發者模式?」,點啟用並輸入密碼。

Note · Missing?
看不到「開發者模式」? LocWarp v0.2.61 起 在狀態列內建一顆「顯示開發者模式選項」按鈕。連上 iPhone 後如果這顆出現,按一下,回 iPhone 把「設定」App 完全關掉再重開,開發者模式選項就會出現。
